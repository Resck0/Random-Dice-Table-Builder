How to ensure you have the latest update:
To be absolutely sure you are using the most recent version of the builder, you can check the following:

Check the Footer/Version: Look for the "Last Updated" date or version number at the bottom of the page (if available).

The "Incognito" Test: Open the site in a New Incognito/Private Window (Ctrl + Shift + N or Cmd + Shift + N).

If the new features or images appear there but not in your normal window, your main browser is definitely stuck on an old cache.

Hard Refresh: If things look "broken" (buttons out of place or missing icons), perform the Force Update steps mentioned above again.

[!NOTE]
GitHub Pages can sometimes take 1 to 3 minutes to deploy your changes after you push your code. If you just saved your changes, wait a moment and refresh!

Why isn't there a "Download Image" button?
Due to modern browser and GitHub security policies, the code is restricted from "reading" image pixels to generate an automatic file. This is a standard security measure to protect user privacy.

Best way to share:
The fastest and most reliable way to share your board is by taking a manual screenshot. This is the standard method used by the Random Dice community to share builds on Discord or WhatsApp, ensuring the best quality and compatibility across all devices.

¿Por qué no hay un botón de "Descargar imagen"?
Debido a las políticas de seguridad de los navegadores modernos y de GitHub, el código no tiene permiso para "leer" los píxeles de las imágenes de los dados para crear un archivo automático. Esto se hace para proteger tu privacidad.
